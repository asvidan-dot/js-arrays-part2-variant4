# Практична робота 7.2: Алгоритми з масивами (Варіант 4)

## Опис проєкту
Цей проєкт містить реалізацію обчислювальних алгоритмів на JavaScript для роботи з масивами без використання вбудованих методів `sort` (де це можливо) або `reverse`.

## Реалізовані алгоритми та аналіз складності (Big O)

| Алгоритм | Часова складність | Просторова складність | Опис |
| :--- | :--- | :--- | :--- |
| **1. Maximum Subarray Sum** | O(n) | O(1) | Алгоритм Кадане для пошуку підмасиву з максимальною сумою. |
| **2. Two Sum Problem** | O(n) | O(n) | Пошук усіх пар із заданою сумою з використанням `Set` для оптимізації. |
| **3. Three Sum Problem** | O(n²) | O(1) | Використання методу двох вказівників після попереднього сортування. |
| **4. Sliding Window Maximum** | O(n) | O(k) | Пошук максимуму у вікні розміру K з використанням структури Deque (черги). |
| **5. Stock Buy-Sell** | O(n) | O(1) | Жадібний алгоритм для пошуку максимального прибутку від купівлі-продажу. |

## Технічні деталі
* Усі алгоритми оптимізовані та не використовують брутфорс (O(n²), де це можливо, і вище).
* Врахована робота з від'ємними числами.
* Реалізовано пошук **всіх** можливих розв'язків для задач Two Sum та Three Sum з відсіюванням дублікатів.

## Тестування та візуалізація
Проєкт розбито на три файли:
* `algorithms.js` — логіка алгоритмів.
* `app.js` — тестові набори даних та візуалізація.
* `index.html` — інтерфейс для виведення результатів.

Алгоритми протестовані на масивах з від'ємними числами, дублікатами та специфічних наборах даних (ціни акцій, вікна).
